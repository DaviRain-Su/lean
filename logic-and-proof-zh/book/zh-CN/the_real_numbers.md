# 第 19 章：实数

> 本译文对应原书 [The Real Numbers](https://leanprover.github.io/logic_and_proof/the_real_numbers.html)；英文 Sphinx 源：`the_real_numbers.rst`。

## 19.1. 数系

我们已经遇到过一些基本数系：自然数 $\mathbb{N}$、整数 $\mathbb{Z}$ 和有理数 $\mathbb{Q}$。在某种意义上，列表中的每一个后续数系都是为了弥补前一个数系的缺陷而设计的。我们可以用任意整数减去任意整数得到另一个整数，也可以用任意非零有理数去除任意有理数得到另一个有理数。

整数满足以下所有性质：

- 加法满足结合律和交换律。
- 存在加法单位元 $0$，且每个元素 $x$ 都有加法逆元 $-x$。
- 乘法满足结合律和交换律。
- 存在乘法单位元 $1$。
- 乘法对加法满足分配律：对任意 $x$、$y$、$z$，有 $x(y+z) = xy + xz$。
- 序关系 $\leq$ 是全序。
- 对任意元素 $x$、$y$、$z$，如果 $x \leq y$，则 $x + z \leq y + z$。
- 对任意元素 $x$、$y$，如果 $0 \leq x$ 且 $0 \leq y$，则 $0 \leq xy$。

前五条说明，配备 $\times$、$+$、$0$、$1$ 后，整数构成一个**交换环**；后三条说明，连同 $\leq$，这个结构是一个**有序环**。自然数缺少加法逆元，因此满足稍弱的一组公理，使其成为一个**有序半环**。另一方面，有理数也构成有序环，并且满足如下额外性质：

- 每个非零元素都有乘法逆元 $x^{-1}$。

这使得它们成为**有序域**的一个实例。

值得了解的是，一旦我们有了自然数，就可以用你已经见过的集合论构造来**构造**整数和有理数。例如，我们可以把一个整数定义为一对自然数 $(i, n)$，其中 $i$ 为 $0$ 或 $1$，意图是 $(0, n)$ 表示正整数 $n$，$(1, n)$ 表示负整数 $-(n+1)$。（我们用 $-(n+1)$ 而不是 $-n$ 来避免 0 有两种表示。）按这个定义，整数就是 $\{0, 1\} \times \mathbb{N}$。然后我们可以继续定义加法、乘法、加法逆元和序关系，并证明它们具有期望的性质。

这种构造有一个副作用：自然数本身不是整数；例如，我们必须区分自然数 2 和整数 2。在 Lean 中就是这样。在普通数学中，通常把自然数看作整数的子集。然而，一旦我们构造出整数，就可以丢弃旧版本的自然数，之后把自然数识别为非负整数。

对有理数也可以做同样的事，把它们定义为 $\mathbb{Z} \times \mathbb{N}_{>0}$ 中的对 $(a, b)$，其中要么 $a = 0$ 且 $b = 1$，要么 $b > 0$ 且 $a$ 与 $b$ 没有公共约数（除了 $1$ 和 $-1$）。思路是 $(a, b)$ 表示 $a/b$。按这个定义，有理数实际上是 $\mathbb{Z} \times \mathbb{N}$ 的一个子集，然后我们可以相应地定义所有运算。

在下一节中，我们将定义一种更复杂的方法，它可以扩展到实数的定义。在后面的章节中，我们还将展示如何从集合论公理构造自然数。这表明我们可以从底部向上构造所有数系。

但首先，让我们 pause 片刻，思考为什么需要实数。我们已经看到 2 没有有理平方根。这意味着，在某种意义上，有理数中存在一个“缺口”：存在平方任意接近 2 的有理数，但没有有理数 $x$ 满足 $x^2 = 2$。但直观上显然应该存在某个具有该性质的**数**：$\sqrt{2}$ 是边长为 1 的正方形的对角线长度。类似地，半径为 1 的圆的面积 $\pi$ 也不在有理数中。这些正是实数被设计来修补的缺陷。

你可能习惯把实数看作（可能无限的）小数：例如，$\sqrt{2} = 1.41421356\ldots$，$\pi = 3.14159265\ldots$。本章的一个中心目标是把“$\ldots$”精确化。思路是，我们可以把一个无限小数看作一个有理逼近序列。例如，我们可以用序列 $1, 1.4, 1.41, 1.414, \ldots$ 逼近 $\sqrt{2}$。我们想把这个序列的“极限”定义为 $\sqrt{2}$，但我们已经看到这个序列在有理数中没有极限。因此，我们必须构造新的对象——实数——来担当这个角色。

事实上，我们将或多或少地把实数**定义**为这样的有理逼近序列。但我们必须处理一个事实：例如，有很多方式可以逼近 $\sqrt{2}$。我们可以从上方逼近：$2, 1.5, 1.42, \ldots$，也可以上下振荡逼近。下一节将展示我们如何把所有这些序列“粘合”在一起，并把它们当作一个对象处理。

## 19.2. 商构造

设 $A$ 是任意集合，$\equiv$ 是 $A$ 上的任意等价关系。回忆[13.3 节](relations.md#133-等价关系与相等)，我们可以把 $A$ 的每个元素 $a$ 指派到其等价类 $[a]$，其中 $b \in [a]$ 表示 $b \equiv a$。这个指派具有如下性质：对所有 $a$、$b$，$a \equiv b$ 当且仅当 $[a] = [b]$。

给定任意集合 $A$ 和等价关系 $\equiv$，定义 $A/\equiv$ 为 $A$ 模 $\equiv$ 的**等价类**组成的集合 $\{ [a] \mid a \in A \}$。这个集合称为“$A$ 模 $\equiv$”，或 $A$ 被 $\equiv$ 除的**商**。你可以把它想象成把 $A$ 中等价的元素“粘合”在一起，形成一个更粗的集合。

例如，如果我们考虑整数 $\mathbb{Z}$，其中 $\equiv$ 表示模 5 等价（如[17.4 节](elementary_number_theory.md#174-模算术)所述），那么 $\mathbb{Z}/\equiv$ 就是 $\{[0], [1], [2], [3], [4]\}$。我们可以在 $\mathbb{Z}/\equiv$ 上定义加法为 $[a] + [b] = [a + b]$。为使这个定义有意义，重要的是知道等式右边不依赖于我们选择的 $[a]$ 和 $[b]$ 的代表元。换言之，我们需要知道：只要 $[a] = [a']$ 且 $[b] = [b']$，就有 $[a+b] = [a'+b']$。而这又等价于说：如果 $a \equiv a'$ 且 $b \equiv b'$，则 $a + b \equiv a' + b'$。换言之，我们要求加法运算**尊重**等价关系；我们在[17.4 节](elementary_number_theory.md#174-模算术)中已经看到这确实成立。

这种把一个定义在集合上的函数转移到该集合的商上的策略由以下定理给出。

---

**定理。** 设 $A$、$B$ 是任意集合，$\equiv$ 是定义在 $A$ 上的任意等价关系，$f : A \to B$。假设 $f$ 尊重等价关系，即对 $A$ 中任意 $a$、$a'$，如果 $a \equiv a'$，则 $f(a) = f(a')$。那么存在唯一的函数 $\bar{f} : A/\equiv \to B$，定义为对所有 $a \in A$，$\bar{f}([a]) = f(a)$。

**证明。** 我们通过把等价类 $x$ 写成 $x = [a]$ 的形式，并令 $\bar{f}(x) = f(a)$，来定义 $\bar{f}$ 在等价类上的值。换言之，我们说 $\bar{f}(x) = y$ 当且仅当存在 $a$ 使得 $x = [a]$ 且 $f(a) = y$。这个定义可疑之处在于，它先验地可能依赖于我们把 $x$ 表示成这种形式的方式；换言之，我们需要证明存在**唯一**的 $y$ 满足这个描述。具体地，我们需要知道：如果 $x = [a] = [a']$，则 $f(a) = f(a')$。但由于 $[a] = [a']$ 等价于 $a \equiv a'$，这正是在说 $f$ 尊重等价关系，而这正是我们的假设。

---

数学家经常通过等式 $\bar{f}([a]) = f(a)$ 来“定义”$\bar{f}$，然后把上面的证明表达为“$\bar{f}$ 是 well defined”的证明。这令人困惑。他们真正想表达的就是定理所说的内容，即存在唯一满足该描述的函数。

为了构造整数，从 $\mathbb{N} \times \mathbb{N}$ 开始。把一对自然数 $(m, n)$ 看作表示 $m - n$，其中减法发生在整数中（我们尚未构造出来！）。例如，$(2, 5)$ 和 $(6, 9)$ 都表示整数 $-3$。直观上，当 $m - n = m' - n'$ 时，对 $(m, n)$ 和 $(m', n')$ 表示同一个整数，但我们还不能这样说，因为我们还没有定义适当的减法概念。不过，这个等式等价于 $m + n' = m' + n$，而**这个**等式在自然数的加法下是有意义的。

---

**定义。** 在 $\mathbb{N} \times \mathbb{N}$ 上定义关系 $\equiv$：$(m, n) \equiv (m', n')$ 当且仅当 $m + n' = m' + n$。

**命题。**$\equiv$ 是等价关系。

**证明。** 自反性：显然 $(m, n) \equiv (m, n)$，因为 $m + n = m + n$。

对称性：假设 $(m, n) \equiv (m', n')$。这意味着 $m + n' = m' + n$。但等式的对称性蕴涵 $(m', n') \equiv (m, n)$，证毕。

传递性：假设 $(m, n) \equiv (m', n')$ 且 $(m', n') \equiv (m'', n'')$。则有 $m + n' = m' + n$ 和 $m' + n'' = n' + m''$。把这两个等式相加，得到

$$
m + n' + m' + n'' = m' + n + n' + m''.
$$

两边减去 $m' + n'$，得到 $m + n'' = n + m''$，这等价于 $(m, n) \equiv (m'', n'')$，证毕。

---

我们现在可以把整数定义为 $\mathbb{N} \times \mathbb{N} / \equiv$。如何定义加法？如果 $[(m, n)]$ 表示 $m - n$，$[(u, v)]$ 表示 $u - v$，那么 $[(m, n)] + [(u, v)]$ 应该表示 $(m + u) - (n + v)$。因此，定义 $[(m, n)] + [(u, v)] = [(m + u, n + v)]$ 是合理的。为使这有效，我们需要知道把 $(m, n)$ 和 $(u, v)$ 映射到 $(m + u, n + v)$ 的运算尊重等价关系。

---

**命题。** 如果 $(m, n) \equiv (m', n')$ 且 $(u, v) \equiv (u', v')$，则 $(m + u, n + v) \equiv (m' + u', n' + v')$。

**证明。** 第一个等价意味着 $m + n' = m' + n$，第二个意味着 $u + v' = u' + v$。把两个等式相加，得到 $(m + u) + (n' + v') = (m' + u') + (n + v)$，这正好说明 $(m + u, n + v) \equiv (m' + u', n' + v')$。

---

每个自然数 $n$ 都可以用整数 $[(n, 0)]$ 表示，特别地，$0$ 用 $[(0, 0)]$ 表示。而且，如果 $[(m, n)]$ 是任意整数，我们可以定义它的负数为 $[(n, m)]$，因为

$$
[(m, n)] + [(n, m)] = [(m + n, n + m)] = [(0, 0)],
$$

由于 $(m + n, n + m) \equiv (0, 0)$。简而言之，我们“发明”了负数！

我们可以继续用这种方式定义整数上的乘法和序关系，并证明它们具有期望的性质。对有理数也可以进行类似的构造。这里，我们从集合 $\mathbb{Z} \times \mathbb{Z}_{>0}$ 开始，其中 $\mathbb{Z}_{>0}$ 表示严格正整数。思路当然是 $(a, b)$ 表示 $a/b$。基于这一点，定义 $(a, b) \equiv (c, d)$ 当 $ad = bc$ 是合理的。然后我们也可以在那里定义加法、乘法和序关系。细节繁琐且不太有启发性，因此我们转而构造实数。

## 19.3. 构造实数

我们面临的问题在于，有理数序列 $1, 1.4, 1.41, 1.414, 1.4142, \ldots$ 似乎趋近于一个“应该”是 2 的平方根的值，但没有有理数能扮演这个角色。下面的定义捕捉了一个有理数序列“似乎趋近于某个值”的概念，而不涉及它正在趋近的那个值。

---

**定义。** 有理数序列 $(q_i)_{i \in \mathbb{N}}$ 称为**柯西序列**（Cauchy sequence），如果对每个有理数 $\varepsilon > 0$，存在某个自然数 $N \in \mathbb{N}$，使得对所有 $i, j \geq N$，都有 $|q_i - q_j| < \varepsilon$。

---

粗略地说，柯西序列是一个元素彼此任意接近的序列，不仅与后继项接近，而且与之后的所有项都接近。在数学中，常用 $\varepsilon$ 表示一个 intended to denote 很小的量；你应该把“对每个 $\varepsilon > 0$”读作“无论 $\varepsilon$ 多么小”。因此，一个序列是柯西序列，如果对于任意小的 $\varepsilon > 0$，都存在某个点 $N$，超过这个点之后元素之间的距离都小于 $\varepsilon$。

柯西序列可以用来描述有理数中的这些缺口，但如上所述，许多柯西序列可以用来描述同一个缺口。在这个阶段，说它们“趋近于同一个点”略有误导，因为没有有理点是它们趋近的对象；更精确的说法是这些序列最终变得任意接近。

---

**定义。** 两个柯西序列 $p = (p_i)_{i \in \mathbb{N}}$ 和 $q = (q_i)_{i \in \mathbb{N}}$ 称为**等价**的，如果对每个有理数 $\varepsilon > 0$，存在某个自然数 $N \in \mathbb{N}$，使得对所有 $i \geq N$，都有 $|p_i - q_i| < \varepsilon$。我们用 $p \equiv q$ 表示 $p$ 与 $q$ 等价。

**命题。**$\equiv$ 是柯西序列上的等价关系。

**证明。** 自反性和对称性容易证明，我们来证明传递性。假设 $(p_i) \equiv (q_i)$ 且 $(q_i) \equiv (r_i)$。我们要证明序列 $(p_i)$ 与 $(r_i)$ 等价。于是，给定任意 $\varepsilon > 0$，选择足够大的 $N_0$，使得对每个 $i \geq N_0$，$|p_i - q_i| < \varepsilon/2$。再选择另一个数 $N_1$，使得对每个 $i \geq N_1$，$|q_i - r_i| < \varepsilon/2$。令 $N = \max(N_0, N_1)$。则对每个 $i \geq N$，有

$$
|p_i - r_i| = |(p_i - q_i) + (q_i - r_i)| \leq |p_i - q_i| + |q_i - r_i| < \varepsilon/2 + \varepsilon/2 = \varepsilon,
$$

证毕。

---

注意，证明中使用了**三角不等式**：对任意有理数 $a$、$b$，$|a + b| \leq |a| + |b|$。如果我们把 $|a|$ 定义为 $a$ 和 $-a$ 中的较大者，那么三角不等式实际上对任何有序环都成立：

---

**定理。** 设 $a$、$b$ 是任意有序环中的元素。则 $|a + b| \leq |a| + |b|$。

**证明。** 根据绝对值的定义，只需证明 $a + b \leq |a| + |b|$ 和 $-(a + b) \leq |a| + |b|$。第一个结论来自 $a \leq |a|$ 和 $b \leq |b|$。对第二个结论，类似地有 $-a \leq |a|$ 和 $-b \leq |b|$，因此 $-(a + b) = -a + (-b) \leq |a| + |b|$。

---

在上面的定理中，如果令 $a = x - y$，$b = y - z$，就得到 $|x - z| \leq |x - y| + |y - z|$。$|x - y|$ 表示数轴上 $x$ 与 $y$ 之间的距离，这解释了它的名字：对任意三个“点”$x$、$y$、$z$，$x$ 到 $z$ 的距离不会大于 $x$ 到 $y$ 的距离加上 $y$ 到 $z$ 的距离。

现在令 $A$ 为有理数的柯西序列组成的集合，并定义实数 $\mathbb{R}$ 为 $A/\equiv$。换言之，实数是有理数柯西序列的集合，模掉我们刚定义的等价关系。

仅仅有集合 $\mathbb{R}$ 还不够：我们还想知道如何对实数进行加、减、乘、除。与整数一样，我们需要先定义底层集合上的运算，然后证明它们尊重等价关系。例如，我们将说明如何加有理数的柯西序列，然后证明如果 $p_1 \equiv p_2$ 且 $q_1 \equiv q_2$，则 $p_1 + q_1 \equiv p_2 + q_2$。然后我们可以通过定义 $[p] + [q] = [p + q]$ 把这个定义提升到 $\mathbb{R}$ 上。

幸运的是，定义柯西序列的加法、减法和乘法很容易。如果 $p = (p_i)_{i \in \mathbb{N}}$ 和 $q = (q_i)_{i \in \mathbb{N}}$ 是柯西序列，令 $p + q = (p_i + q_i)_{i \in \mathbb{N}}$，减法和乘法类似。要证明这些序列本身也是柯西序列，并证明这些运算具有适当的代数性质，则比较棘手。我们在练习中请你证明其中一些性质。

我们可以把每个有理数 $q$ 与常值柯西序列 $q, q, q, \ldots$ 等同起来，因此实数包含有理数。下一步是抽象掉我们所选构造的细节，以便从此可以抽象地使用实数，而不再把它们看作由有理数柯西序列的等价类给出。

## 19.4. 实数的完备性

我们构造实数是为了填补有理数中的缺口。怎么知道我们已经把它们全部补上了呢？也许我们需要用实数的柯西序列构造更多的数？下面的定理告诉我们，恰恰相反，用这种方式再也不需要扩展实数了。

---

**定义。** 设 $r$ 是实数。实数序列 $(r_i)_{i \in \mathbb{N}}$ **收敛**于 $r$，如果对每个 $\varepsilon > 0$，存在 $N$，使得对每个 $i \geq N$，$|r_i - r| < \varepsilon$。

**定义。** 序列 $(r_i)_{i \in \mathbb{N}}$ **收敛**，如果它收敛于某个 $r$。

**定理。** 每个实数的柯西序列都收敛。

---

这条定理的陈述常被称为“实数是**完备**的”。粗略地说，它意味着无论你在哪里寻找一个实数，你都一定会找到一个。下面是类似的原则。

---

**定义。** 元素 $u \in \mathbb{R}$ 称为子集 $S \subseteq \mathbb{R}$ 的一个**上界**，如果 $S$ 中每个元素都小于或等于 $u$。如果 $S$ 存在上界，则称 $S$ **有上界**。元素 $u$ 称为 $S$ 的**最小上界**，如果它是 $S$ 的上界，并且任何比 $u$ 小的数都不是 $S$ 的上界。

**定理。** 设 $S$ 是 $\mathbb{R}$ 的有上界的非空子集。则 $S$ 有最小上界。

---

有理数没有这条性质：如果我们令 $S = \{ x \in \mathbb{Q} \mid x^2 < 2 \}$，那么有理数 2 是 $S$ 的一个上界，但 $S$ 在有理数中没有最小上界。

一个基本定理指出，实数恰好由如下性质刻画：它们是完备有序域，并且每个实数 $r$ 都小于或等于某个自然数 $N$。任何两个满足这些要求的模型在涉及常数 $0$、$1$、运算 $+$ 和 $\cdot$ 以及关系 $\leq$ 的行为上必须完全相同。这个事实非常强大，因为它使我们能够在正常数学中避免思考柯西序列构造。一旦我们证明了自己的构造满足这些要求，我们就可以把 $\mathbb{R}$ 当作“那个”唯一的完备全序域，忽略任何实现细节。我们还可以自由选择任何方式实现 $\mathbb{R}$，只要它满足这个接口；并且只要定理不涉及底层表示，它们对所有构造都同样成立。

## 19.5. 另一种构造

许多资料使用另一种构造实数的方法，把它们看作**戴德金分割**（Dedekind cuts）。戴德金分割是有理数集合的有序对 $(A, B)$，满足以下性质：

- 每个有理数 $q$ 要么在 $A$ 中，要么在 $B$ 中。
- 每个 $a \in A$ 小于每个 $b \in B$。
- $A$ 中没有最大元素。
- $A$ 和 $B$ 都非空。

前两条性质说明为什么我们把这对集合称为“分割”：集合 $A$ 包含数轴上某个标记左侧的所有有理数，$B$ 包含右侧的所有点。第三条性质告诉我们标记处发生了什么。但有两种可能：要么 $B$ 有最小元素，要么没有。想象 $A$ 没有最大元素且 $B$ 没有最小元素的情形可能有点 tricky，但考虑例子 $A = \{ x \in \mathbb{Q} \mid x^2 < 2 \}$ 和 $B = \{ x \in \mathbb{Q} \mid x^2 > 2 \}$。不存在有理数 $q$ 使得 $q^2 = 2$，但两边都有任意接近的有理数；因此 $A$ 和 $B$ 都不包含端点。

我们可以把 $\mathbb{R}$ 定义为戴德金分割的集合。戴德金分割 $(A, B)$ 对应于有理数 $q$，如果 $q$ 是 $B$ 的最小元素；如果 $B$ 没有最小元素，则对应于无理数。在 $\mathbb{R}$ 上定义加法很直接：

$$
(A_1, B_1) + (A_2, B_2) = (\{ a_1 + a_2 \mid a_1 \in A_1, a_2 \in A_2 \}, \{ b_1 + b_2 \mid b_1 \in B_1, b_2 \in B_2 \}).
$$

一些作者更喜欢这种构造而不是柯西序列构造，因为它避免了取集合的商，从而省去了证明算术运算尊重等价关系的麻烦。另一些作者更喜欢柯西序列，因为它提供了更清晰的逼近概念：如果实数 $r$ 由柯西序列 $(q_i)_{i \in \mathbb{N}}$ 给出，那么对足够大的 $N$，$q_N$ 就给出了 $r$ 的任意接近的有理逼近。

不过，对大多数数学家的大多数时间来说，差别无关紧要。两种构造都创造出完备线性有序域，并且在某种意义上，它们创造出**同一个**完备线性有序域。严格地说，柯西实数的集合不等于戴德金实数的集合，因为一者由有理数柯西序列的等价类组成，另一者由有理数集合的对组成。但两个集合之间存在一个保持域性质的双射。也就是说，存在从柯西实数到戴德金实数的双射 $f$，使得

- $f(0) = 0$
- $f(1) = 1$
- $f(x + y) = f(x) + f(y)$
- $f(x \cdot y) = f(x) \cdot f(y)$
- $f(-x) = -f(x)$
- $f(x^{-1}) = f(x)^{-1}$
- $f(x) \leq f(y) \iff x \leq y$

我们说这两种构造是**同构的**，函数 $f$ 是一个**同构**。由于我们通常只关心实数作为完备有序域的地位，而两种构造作为有序域不可区分，使用哪种构造都没有关系。

## 19.6. 练习

1. 证明[19.2 节](the_real_numbers.md#192-商构造)中定义的整数加法是交换且结合的。

2. 从[19.2 节](the_real_numbers.md#192-商构造)中整数的构造出发，证明对每个整数 $a$ 有 $a + 0 = a$。

3. 通过 $a - b = a + (-b)$ 定义整数的减法，并证明对每对整数 $a$、$b$ 有 $a - b + b = a$。

4. 定义整数的乘法：先在底层表示上定义，然后证明该运算尊重等价关系。

5. 证明每个柯西序列都有界：即如果 $(q_i)_{i \in \mathbb{N}}$ 是柯西序列，则存在某个有理数 $M$，使得对所有 $i$ 有 $|q_i| \leq M$。提示：尝试令 $\varepsilon = 1$。

6. 设 $p = (p_i)_{i \in \mathbb{N}}$ 和 $q = (q_i)_{i \in \mathbb{N}}$ 是柯西序列。定义 $p + q = (p_i + q_i)_{i \in \mathbb{N}}$，$p q = (p_i q_i)_{i \in \mathbb{N}}$。

   1. 证明 $p + q$ 是柯西序列。即对任意 $\varepsilon > 0$，证明存在 $N$，使得对所有 $i, j \geq N$，$|(p_i + q_i) - (p_j + q_j)| < \varepsilon$。
   2. 证明 $p q$ 是柯西序列。除了三角不等式，你会发现上一题也很有用。

7. 以下两部分说明柯西序列的加法尊重等价。

   1. 证明如果 $p$、$p'$、$q$ 是柯西序列且 $p \equiv p'$，则 $p + q \equiv p' + q$。
   2. 利用本题的上一部分证明：如果 $p$、$p'$、$q$、$q'$ 是柯西序列，$p \equiv p'$ 且 $q \equiv q'$，则 $p + q \equiv p' + q'$。你可以使用实数上加法交换这一事实。

8. 证明如果 $(A_1, B_1)$ 和 $(A_2, B_2)$ 是戴德金分割，则 $(A_1, B_1) + (A_2, B_2)$ 也是戴德金分割。
