01 · Setup

安装和项目结构

官网推荐 VS Code/Cursor + Lean 4 扩展。命令行由 elan 管理 Lean 版本，lake 管理项目。

安装工具链

macOS 上安装 Lean 版本管理器 elan，然后加载环境变量。

curl https://elan.lean-lang.org/elan-init.sh -sSf | sh
source ~/.elan/env

安装编辑器扩展

Lean 的交互体验主要靠 InfoView：目标、假设、错误都会实时显示。

code --install-extension leanprover.lean4

创建项目

所有正式 Lean 代码都建议放进 Lake 项目。

lake new MyLeanProject
cd MyLeanProject
lake build
code .

03 · Proof notes

证明笔记：从计算到归纳

Natural Number Game 的重点不是“Lean 会算”，而是把每一步改写暴露出来。

2 + 2 = 4 
theorem two_add_two : 2 + 2 = 4 := by
  rfl

标准 Lean 里 rfl 可以直接通过。游戏里会要求你用 rw 展开 succ，训练“等式改写”的手感。

归纳假设 hd

证明 zero_add 时，归纳步骤里有：

hd : 0 + d = d

当目标里出现 0 + d，就可以：

rw [hd]

它只是把当前目标里的 0 + d 替换成 d。

succ_add 关键点 
theorem succ_add (a b : ℕ) : succ a + b = succ (a + b) := by
  induction b with d hd
  rw [add_zero]
  rfl
  rw [add_succ]
  rw [add_succ]
  rw [hd]
  rfl

卡点通常是少了第二个 rw [add_succ]：左边和右边都要把加法展开到能看见归纳假设的位置。

05 · Cloudflare Pages

部署到 Cloudflare

这个站是纯静态网站，最适合 Cloudflare Pages。后面登录 Cloudflare 后，用 Wrangler 一条命令即可发布。

  1. 确认已登录：npx wrangler whoami
  2. 如果没登录：npx wrangler login
  3. 部署当前目录：npx wrangler pages deploy . --project-name lean-learning-site
# 本地预览
python3 -m http.server 8787
